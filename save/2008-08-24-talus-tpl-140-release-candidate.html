EDIT :: La finale est sortie, à la date prévue :)

Bonsoir à tous,

Voilà, annonce pour Talus' TPL : La version 1.4.0RC (comme Release Candidate) est disponible.

Pourquoi une RC, et non pas une finale ? Cette version ayant de gros changements (passage de la gestion du cache en objet, par exemple), je souhaite avant tout la tester de long de large et en travers. Ne pouvant pas trouver tous les bugs possibles, je m'en remets donc à vous pour les dénicher :)

Signalez-moi les bugs que vous trouvez dans les commentaires SVP :). Si le tout est considéré comme "stable" pendant 2 3 jours, je mettrais cette release en tant que release finale.

Voici le lien de téléchargement (attention, c'est une version zip... !) : Talus' TPL v1.4.0RC.

Bons tests :)
