Bonsoir à tous,

Alors que depuis quelques jours, j'ai lançé la chasse au bugs pour la dernière release de Talus' TPL, je passe dans la dernière étape : le test par d'autres développeurs. Ayant mis la toute dernière version sur Talus' TPL, les sources pour le moteur TPL et le cache FTP sont donc accessibles sur le site.

Avant de publier la release officielle, je souhaiterais avoir quelques retour, pour avoir des reports d'éventuels bugs. Si d'ici la fin de la semaine prochaine, je n'ai pas de bugs "trop" grave (ou alors qu'ils sont réglés), alors cette release sera officielle (via une news sur le site principal). Pour reporter un bug, faites le via les commentaires :). Cette "opération" est surtout lancée pour voir si y'a encore l'erreur du $end pour tous vos TPLs, lorsqu'ils sont compilés une première fois...

Si vous avez des commentaires à faire, n'hésitez pas ;)
